Sobre

Este website foi desenvolvido como parte do trabalho de conclusão de curso (TCC) de dois estudantes do curso de ciência da computação. O propósito do website é servir como uma plataforma para a observabilidade do status de balneabilidade das praias do Rio de Janeiro através da visualização dos dados relativos às correspondentes praias no formato de um mapa interativo e de lista. Ademais, o website também permite o cadastro de seus usuários para que façam a subscrição de suas praias favoritas. Ao se subscrever nas praias desejadas, caso o status de balneabilidade dessas praias mude, os usuários cadastrados receberão um e-mail contendo informações relativas a essa mudança.


Este website foi estruturado da seguinte forma:


  • Mapa: A seção de Mapa permite a visualização dos status de balneabilidade das praias em formato de um mapa interativo;
  • Lista de Praias: A seção de Lista de Praias permite a visualização dos status de balneabilidade das praias em formato de uma lista;
  • Subscrições: A seção de Subscrições permite ao usuário se subscrever nas praias desejadas para receber e-mails contendo informações dos status de balneabilidade das correspondentes praias;
  • Sobre: A seção Sobre contém informações pertinentes sobre a organização e estrutura deste website;
  • Login: A seção Login permite ao usuário já cadastrado realizar login no website;
  • Registro: A seção Registro permite ao usuário não cadastrado realizar o cadastro no website.
